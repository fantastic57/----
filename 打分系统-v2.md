# 打分系统 v2 使用说明

> 2026-07-13 起生效，内置于 `d:\生活\2026买房\买房地图\map.html`

---

## 快速开始

1. 启动服务器：`cd d:\生活\2026买房\买房地图 && python server.py`
2. 浏览器打开 `http://localhost:8080/map.html`
3. 点击地图上任意小区 marker → 详情面板弹出
4. 在评分区域内填字段值，**得分自动计算**，总分实时更新

---

## 10 个评分字段

| 字段 | 输入类型 | 自动打分规则 |
|---|---|---|
| 学区质量 | 下拉 好/一般/差 | 5 / 2.5 / 0 |
| 地铁线路 | 多选 3/8/19/20 | 3号线=2.5，其他各=0.83，全选=5 |
| 三号线步行 | 数字（分钟） | ≤5=5, 5-10=3.33, 10-15=1.67, >15=0 |
| 最近地铁步行 | 数字（分钟） | ≤5=5, 5-10=3.33, 10-15=1.67, >15=0 |
| 学校骑行 | 数字（分钟） | ≤15=5, 15-20=3.33, 20-25=1.67, >25=0 |
| 上班公交 | 数字（分钟） | ≤75=5, 75-90=3.33, 90-105=1.67, >105=0 |
| 上班开车 | 数字（分钟） | ≤50=5, 50-60=3.33, 60-70=1.67, >70=0 |
| 停车 | 下拉 随便停/稍微找/难停 | 5 / 2.5 / 0 |
| 小区环境 | 下拉 美丽家园/中规中矩/差 | 5 / 2.5 / 0 |
| 周边配套 | 下拉 齐全/一般/缺 | 5 / 2.5 / 0 |

## 权重

- 每个字段权重的下拉选项 **1-5**
- 默认所有字段权重 = **3**（等权）
- **全局共享**：改某个小区的权重 → 所有小区同步
- 权重存于 data.json 顶层，不导入/导出

---

## 总分公式

```
总分 = Σ(权重 × 自动得分) / Σ(权重) × 2
```

- 全最佳：10
- 全最差：0
- 权重 5 的项对总分影响是权重 1 项的 5 倍

---

## 价格追踪

挂牌均价、前挂牌均价 → **手动输入**  
跌幅比 → **自动算**：`跌幅比 = (前 - 现) / 前 × 100%`

---

## 数据存储

- 所有小区数据存 `data.json`
- 新格式：`{weights: {...}, markers: [...], others: [...]}`
- 权重修改 → GET/POST 到 `/api/weights`
- 完整数据 → GET/POST 到 `/api/data`
- 不再使用 localStorage
